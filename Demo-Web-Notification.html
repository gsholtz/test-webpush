Link a contact to a notification - onclick="DI.WebPush.Identify("f_EMail", "Valeur");"
Start by adding the contact to send the test to in the Dialog Insight project where Web notifications are configured, type the email address of this contact here and click "Link". You will then be able to create a Web notification in Dialog Insight and send a test to this contact.
